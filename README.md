# AI-Based Autonomous Navigation System

A beginner-friendly autonomous navigation project built using **Python, NumPy, and Matplotlib**.

This project simulates an autonomous agent navigating through a street-like environment using the **A\*** path planning algorithm.

---

## 🚀 Project Overview

The goal of this project is to simulate how an autonomous system can move from a **start position** to a **goal position** while finding the shortest valid route in a virtual environment.

This project demonstrates core concepts used in:

- Autonomous Vehicles
- Robotics
- Path Planning
- Smart Mobility Systems
- AI-based Navigation

---

## 📌 Features

- Street-like grid environment
- Start and goal positions
- A* shortest path planning
- Agent movement simulation
- Matplotlib-based visualization
- Beginner-friendly modular project structure

---

## 🛠 Tech Stack

- **Python**
- **NumPy**
- **Matplotlib**
